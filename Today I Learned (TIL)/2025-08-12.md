강화학습 4장 [[4. Model-Free Prediction]] 복습

prediction은 MDP를 모를 때 value function을 찾는 문제

즉, $$ v_\pi(s) = \mathbb{E}_\pi[G_t|S_t = s] $$
상태 $s_t$에서의 return의 기댓값을 구하는 것

return의 정의는 다음과 같았음

$$
G_t = R_{t + 1} + \gamma R_{t + 1} + \ldots + \gamma^{T - 1} R_T
$$

Monte-Carlo / Temporal Difference 방법이 존재

- Monte-Carlo: 실제로 rollout을 하면서 받은 return을 상태별로 저장했다가 이를 평균해서 계산
- Temporal Difference
	- 1 step만 보는 TD(0): $R_{t + 1} + \gamma V(S_{t + 1})$로 업데이트 (즉, 추정치로 업데이트)
		- 실제 값 $R_{t + 1}$이 반영됐기 때문에, 조금씩 실제 value function으로 수렴함
	- n step 보는 TD($\lambda$)
		- forward-view TD($\lambda$): n-step이 끝날 때까지 업데이트 하지 못함. (끝나야 return을 알 수 있으므로)
		- backward-view TD($\lambda$): n-step이 끝나지 않아도 업데이트 가능.
			- Eligibility trace: 상태에 방문할 때마다, 1씩 증가시키고 방문하지 않을 때마다 $\gamma$만큼 eligibility trace 값을 감소시킴
			- TD(0)의 업데이트 식에다가 eligibility trace를 곱하기만 하면, TD($\lambda$)와 동일한 결과를 얻을 수 있음.
			- $V_t(S_t) \leftarrow V_{t - 1}(S_t) + \alpha \delta_t E_t(S_t)$

- Bootstrapping: 추정치로 업데이트
	- DP, TD
- Sampling: 샘플링을 통해 기댓값을 계산
	- MC, TD
