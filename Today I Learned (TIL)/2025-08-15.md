강화학습 6장 Value function approximation

이전까지는 table lookup 방식으로, 모든 상태(혹은 상태-행동 쌍)에 대해 테이블을 만들어서 업데이트 했었다.

하지만 현실의 많은 문제들은 테이블로 관리하기에는 너무 커서 function approximator로 이를 대체하려는 시도들이 있었다.

function approximator에는 linear combination, non-linear function 등 다양한 방법들이 존재하는데(gradient를 구해서 업데이트하기 위해 미분가능한 함수들만 사용) 강의에서는 설명의 편의를 위해 linear combination으로 설명한다. (요새는 거의 neural network만 사용하는 것 같다)

기본적인 아이디어는 다음과 같다.

1. MSE(Mean Squared Error)로 target과의 차이를 줄이는 방향으로 업데이트 한다.
2. 이때 target 값으로 이전 강의에서 공부한 MC, TD(0), TD($\lambda$)를 쓸 수 있다.
3. Generalized Policy Iteration을 반복한다. (policy improvement: $\epsilon$-greedy policy improvement)

non-linear function을 사용할 경우 수렴하지 않는다는 문제가 있을 수 있지만, 이론적으로 그럴 뿐 실제로는 대게 잘 된다고 한다.

마지막에 DQN에 대해 설명하는데, 네이처에도 실렸던 논문이라 중요한 논문이라는 것은 알고 있었지만 오늘 다시 강의를 복습하면서 왜 중요한 논문인지를 깨닫게 되었다
DQN은 off-policy, TD(0), non-linear function approximator라는 수렴하기 매우 어려운 조건에서 Atari 게임을 super-human 수준으로 플레이하는 정책을 학습했다.

- 업데이트를 안정적으로 하기 위해 fixed target을 사용
- off-policy를 위해 replay buffer를 사용
